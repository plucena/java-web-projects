WELCOME TO BOOGLE

Adding a new document: 
        
curl -X PUT http://192.241.171.69:8080/Boogle/document/index -H 'Content-type:application/json' -d '
{
    "pageID" : "1",
    "content": "the quick brown fox jumped over the lazy dog"
  }
'

Listing all existing documents 
curl -X GET http://192.241.171.69:8080/Boogle/document/index

Search documents from a web browser 
http://192.241.171.69:8080/Boogle/document/search?query=fire fox
